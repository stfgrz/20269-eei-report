# Output Directory

This directory contains all outputs generated by the analysis.

## Subdirectories

### `/figures/`
Generated graphs and maps:
- Firm size density plots (2007, 2017)
- TFP distribution comparisons (WRDG vs LP)
- China shock maps by NUTS-2 region
- Manufacturing share maps
- RD plots and coefficient plots

### `/tables/`
Generated tables in LaTeX and HTML:
- Descriptive statistics (Tables 1-2)
- Production function coefficients comparison
- China shock regression results
- IV first stage results

### `/intermediate/`
Intermediate datasets created during analysis:
- China shock calculations by region-year
- Merged datasets for regressions
- Cleaned ESS survey data
- NUTS-2 geographic data for mapping

## Reproducing Outputs
All outputs are generated by running `code/main_analysis.do`.
Existing outputs will be overwritten.
