# Data Directory

This directory contains all datasets used in the analysis, organized into raw (original) and processed (generated) subdirectories.

## Directory Structure

```
data/
├── raw/                    # Original datasets (DO NOT MODIFY)
│   ├── shapefiles/        # NUTS-2 2013 shapefiles
│   └── *.dta, *.pdf       # Data files and documentation
└── processed/             # Generated intermediate datasets
    └── *.dta              # Files created by main_analysis.do
```

## Raw Data (`raw/`)

### Firm-Level Data
- **`EEI_TH_2025.dta`** - Amadeus balance sheet data for France, Italy, and Spain (2006-2017)
  - Contains: firm ID, country, NUTS-2 region, sector, sales, capital, labor, materials, value added
  - Used in: Problems 1-4 (productivity analysis)

- **`EEI_TH_P6_2025.dta`** - Extended firm-level dataset
  - Used in: Problem 6 (trade shock and firm outcomes)

### Trade Data
- **`Imports_China_Take_Home.dta`** - Chinese import data by sector and year
  - Source: UN Comtrade
  - Used in: Problem 5 (China shock construction)

- **`Imports_US_China_Take_Home.dta`** - US imports from China (instrumental variable)
  - Source: UN Comtrade
  - Used in: Problem 5 (IV identification strategy)

### Employment Data
- **`Employment_Shares_Take_Home.dta`** - Employment by NUTS-2 region, sector, and year
  - Contains: employment shares, total employment by region
  - Used in: Problems 5-6 (computing China shock weights)

### Survey Data
- **`ESS8e02_3.dta`** - European Social Survey Round 8 (2016)
  - Individual-level data on attitudes, demographics, and political preferences
  - Used in: Problem 7 (political economy analysis)

- **`ESS8_cleaned.dta`** - Cleaned version of ESS8
  - Pre-processed for analysis

- **`ESS8e02_3 codebook.html`** - ESS8 data documentation

### Political Data
- **`MPDataset_MPDS2024a_stata14.dta`** - Manifesto Project Dataset
  - Party manifestos and political positions
  - Used in: Problem 7 (party-level analysis)

- **`codebook_MPDataset_MPDS2024a.pdf`** - Manifesto Project documentation
- **`handbook_2021_version_5.pdf`** - Manifesto Project handbook

### Spatial Data (`shapefiles/`)
- **`NUTS_RG_20M_2013_3035.*`** - Eurostat NUTS-2 2013 shapefiles
  - Format: .shp, .dbf, .shx, .prj, .cpg
  - Projection: EPSG:3035 (LAEA Europe)
  - Used in: Problems 5-6 (regional mapping)

## Processed Data (`processed/`)

This subdirectory contains intermediate datasets generated by `code/main_analysis.do`. These files should NOT be committed to version control (they are listed in `.gitignore`).

### Generated Files (examples)
- `ChinaShock_by_region_year.dta` - China shock by NUTS-2 region and year
- `sum_china_shock_merged.dta` - Collapsed China shock measures
- `nuts2_db.dta` - NUTS-2 shapefile database
- `nuts2_coords.dta` - NUTS-2 coordinates for mapping
- `weights_pre.dta` - Employment weights for shock calculation
- `part_I_cleaned_sample.dta` - Cleaned firm-level data
- And others...

## Data Sources

1. **Amadeus (Bureau van Dijk)** - Firm-level balance sheet data
2. **UN Comtrade** - International trade statistics
3. **European Social Survey (ESS)** - Individual-level survey data
4. **Manifesto Project** - Political party manifestos
5. **Eurostat** - NUTS regional shapefiles

## Usage Notes

### For Raw Data
- **DO NOT modify** files in `raw/` - they should remain unchanged
- Some data files are loaded directly from GitHub URLs in the code
- Codebooks and documentation files (`.pdf`, `.html`) provide variable descriptions

### For Processed Data
- Files in `processed/` are automatically generated when running the analysis
- These files are excluded from version control via `.gitignore`
- To regenerate: delete the files and rerun `code/main_analysis.do`

## Data Citation

## Notes

- All monetary values are in real terms (deflated)
- NUTS-2 regions follow the 2013 definition
- Time period: primarily 2006-2017 for firm data, 1988-2007 for trade shocks
