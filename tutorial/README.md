# Tutorial Directory

This directory contains STATA tutorial materials used in the course.

## Contents

### `Tutorial_EEI.do`
STATA do-file with tutorial exercises covering:
- Basic STATA commands and syntax
- Data manipulation and management
- Statistical analysis techniques
- Regression analysis
- Data visualization

### `STATA_Tutorial_EEI.dta`
Practice dataset used in the tutorial exercises.

## Purpose

These files are provided as:
- Learning materials for STATA beginners
- Reference for STATA syntax and commands
- Practice exercises complementing the course

## Usage

To work through the tutorial:
```stata
do tutorial/Tutorial_EEI.do
```

Make sure the working directory is set to the repository root.

## Note

These tutorial files are **separate from the main analysis** and are not required for the take-home exam submission. They are included for completeness and as learning resources.
